

                        SIO2IDE for ATARI
                     Version 4.3 (23.06.2004)


 Package contents:

  Documents:
    \DOCS
        S2I_PCB.GIF       - PCB layout (gif format)
        S2I_SCH.GIF       - schematic (gif format)
        Tech_Manual...    - technical specification (text format)
        Users_Manual...   - users manual (text format)

  Atari Software:
    \SOFT\ATARI\FDISK
                          - Atari fdisk.com software sources
                            to rebuild the software type 'make all'
    \SOFT\ATARI\FDISK\BIN
                          - fdisk.com executable for Atari DOS
                          - fdisk.rom executable for Cartridge (XEGS 32kB)

  AVR Software:
    \SOFT\AVR\
                          - common makefiles for AVR software
        Config.MAK        - edit CMPLR entry if needed (default is IARAVR)
    \SOFT\AVR\GNUAVR
                          - makefiles for GNUAVR compiler
    \SOFT\AVR\IARAVR
                          - makefiles for IARAVR compiler
    \SOFT\AVR\FAT_1632
                          - AVR software sources
                            to rebuild the software type 'make all'
    \SOFT\AVR\FAT_MM\BIN
                          - sio2ide.hex (Intel Hex file) for AVR
                          - to programm AVR micro run 'prg.bat'

  PC Software:
    \SOFT\PC\CHECKFS
                          - checkfs.exe software sources
                            to rebuild the software type 'make all'
    \SOFT\PC\CHECKFS\BIN
                          - checkfs.exe executable
    \SOFT\PC\MAKEATR
                          - makeatr.exe software sources
                            to rebuild the software type 'make all'
    \SOFT\PC\MAKEATR\BIN
                          - makeatr.exe executable
    \SOFT\PC\WIN98drv
                          - USB drivers for Win98
