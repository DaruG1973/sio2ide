
SIO2IDE schematics and PCB (Protel format).

   sio2ide.sch          - root SIO2IDE schematic
   sio2ide1.sch         - SIO2IDE 1.X schematic with options
   sio2ide2.sch         - SIO2IDE >2.1 schematic with options
   sio2ide.pcb          - SIO2IDE PCB based on the root schematic
   sio2ide.lib          - SIO2IDE PCB library
   sio2ide4.sch         - SIO2IDE 4.0 schematic
   sio2ide4.pcb         - SIO2IDE 4.0 PCB
   usbpcb.lib           - SIO2IDE 4.0 PCB library
   usbsch.lib           - SIO2IDE 4.0 schematic library
